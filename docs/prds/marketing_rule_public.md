# Horus 对外发布文案（合规版）

## 1. 官网长版（建议用于落地页）

### Horus 托管理财

Horus 提供三类托管理财策略：保守型、稳健型、激进型，覆盖从短周期到长周期的多种投资期限。用户通过钱包签名即可完成订阅，并在“我的托管仪表板”中实时查看持仓状态、净值表现与结算结果。

### 产品机制

- 策略类型：保守型 / 稳健型 / 激进型
- 可选期限：1 / 3 / 7 / 15 / 30 / 60 / 90 / 180 / 365 天
- 收费方式：仅在盈利时收取绩效费（不同产品费率不同）
- 结算方式：支持到期结算，也支持符合条件的主动赎回
- 数据透明：支持净值（NAV）和历史记录追踪

### 风险与保障说明

- 保守型产品支持“最低保障机制”，在满足条款时由储备金机制补差。
- 其他策略不提供最低收益保障。
- 所有策略均存在市场波动风险，历史表现不代表未来收益。

### 联盟激励体系

Horus 联盟当前采用 5 级体系，并基于“零号线 + 太阳线”进行激励分配。系统支持团队业绩统计、返佣明细和提现流程，适合长期运营型推广者。

## 2. 社群短版（建议用于 TG/推特/社媒）

Horus 托管理财已上线：

- 三类策略：保守 / 稳健 / 激进
- 多期限选择：1 到 365 天
- 盈利才收绩效费
- 支持净值追踪、到期结算与历史记录
- 保守策略支持最低保障机制（按条款触发）

联盟系统已支持 5 级激励模型（零号线 + 太阳线）。

风险提示：数字资产波动较大，历史表现不代表未来收益。

## 3. KOL 口播版（30 秒）

Horus 现在的托管理财可以简单理解为三档策略加多种期限选择。你用钱包签名就能订阅，后续在仪表板里看净值、看结算，逻辑很清楚。平台是盈利才收绩效费，保守策略还有最低保障机制，但要按产品条款触发。联盟这块目前是 5 级模型，做团队推广也有完整返佣路径。注意，所有策略都有市场风险，过往收益不代表未来。

## 4. FAQ（客服统一回复模板）

### Q1：你们是固定收益吗？

不是。Horus 不承诺固定收益，页面展示的是目标区间和历史表现，仅供参考。

### Q2：是不是亏损也会收费？

不会。当前托管理财是“仅在盈利时收取绩效费”。

### Q3：保底是所有产品都有吗？

不是。仅部分保守型产品有最低保障机制，且以具体产品条款为准。

### Q4：现在是 V1-V9 还是 5 级联盟？

当前线上是 5 级联盟体系，不是 V1-V9。

### Q5：托管理财和代理 tier 是一个收费系统吗？

不是。托管理财是绩效费口径；代理 tier（STARTER/PRO/WHALE）是另一条业务线。

## 5. 禁用词与高风险表述（运营必看）

以下词汇禁止在对外文案中直接使用：

- 稳赚
- 保收益
- 固定回报
- 无风险
- 躺赚

建议替换为：

- 目标区间
- 风险收益特征
- 历史表现仅供参考
- 以产品条款和实际结算为准
