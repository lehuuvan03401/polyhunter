# Optimized Strategy Card Spec

## MODIFIED Requirements

### Requirement: visual-hierarchy
The UI MUST present strategy details with clear visual hierarchy, grouping related information (Risk Settings) together.

#### Scenario: Scanning Strategies
Given the user is viewing the list of strategies
When they look at a card
Then they SHOULD immediately identify the Risk Settings via icons and grouped layout
AND the labels and values MUST be visually connected (close proximity).
